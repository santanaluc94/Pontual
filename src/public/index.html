Este projeto é um relógio de ponto, que permite inserir apontamento diário, edição de apontamentos anteriores (apenas do mês atual), dados do usuário, histórico de apontamentos, solicitação de férias (não podem aplicar apontamentos nos dias marcados como férias) e programação de dias não uteis (incluindo feriados, eventos da empresa, entre outras datas especiais).

A ideia do projeto é contemplar os objetivos acima, podendo ser extendido ou modificado conforme a necessidade de nova(s) funcionalidade(s), ou remoção(ões) de alguma(s) funcionalidade(s) já definida(s), ou sugestão de alguma funcionalidade sugerida pela comunidade.

O objetivo do projeto é o aperfeiçoamento pessoal do desenvolvedor nas práticas de Javascript, aprendizagem do framework Node.js e da ferramenta Gulp.

O projeto ainda está em desenvolvimento.

O manual será incrementado conforme as funcionalidades forem criadas.

Para o desenvolvimento do backend esté projeto será desenvolvido em Node.js utilizando um banco de dados MongoDB para o armazenamento de dados.

Para a realização do frontend está sendo utilizado HTML, Javascript e Less (sendo utilizado o Gulp para o processamento do CSS em Less).

O projeto está disponível nesse Github para qualquer um que queria utilizar um relógio de ponto, podendo ser utilizado individualmente ou para uma empresa.
